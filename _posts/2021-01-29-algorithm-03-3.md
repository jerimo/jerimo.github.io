# 03-3| 숫자 카드 게임

![03-3문제](/assets/image/03-3-1.PNG)
![03-3문제](/assets/image/03-3-2.PNG)

<small>
소요시간: 약 5분
</small>

### 0. 초기 코드:
```c++
#include <iostream>
#include <list>
using namespace std;


int main() {
	int n, m = 0;
	cin >> n >> m;
	int min = 0;
	list<int> li;
	int x = 0;
	for (int i = 0; i < n; i++) {
		for (int j = 0; j < m; j++) {
			cin >> x;
			li.push_back(x);
		}
		li.sort();
		if (min < li.front()) {
			min = li.front();
		}
		li.clear();
	}
	cout << min;

	return 0;
}
```

### 1. 개선할 점:
* 파이썬 같은 경우는 2중 for문을 사용하지 않고 list를 한 줄씩 입력 받아 비교한다.
<br>
<small>이 문제의 경우 입력으로 들어오는 수가 10,000이하이므로 2중 for문을 사용해도 크게 문제가 되지 않는다.</small>
